# 注册一个专属的eth名字

ENS，全称Ethereum Name Service，也就是以太坊名字服务，这个17年就启动的古老项目，最近又突然变身网红，原因是它发币了，并且，所有在11/1日前持有ENS域名的用户，都可以获得空投。

因为深感以太坊地址使用不便，我在9月底注册了一个ENS域名`liaoxuefeng.eth`，结果意外获得239个ENS空投。

![rich](ha.png)

ENS域名服务其实原理非常简单。和DNS一样，普通人是记不住`12.34.56.78`这样的IP地址的，于是有了DNS服务器负责把`abc.xyz`解析成IP，这样我们在上网的时候才能直接输入域名。

在以太坊上，使用地址`0xf52Bb9ea6c029abB54aEe518109bEB8872ab6FF6`真的谁也记不住，但如果用`liaoxuefeng.eth`，不但记得住，而且防止了粘贴出错，对方手抖出错的麻烦。

所以，ENS就是这么一个去中心化名字服务：把一个`*.eth`域名与一个地址关联起来记录在ETH合约里，当然也可以把该域名重新指向一个新地址，这样对方不用关心到底地址变没变。

目前，几乎所有的钱包软件都支持ENS，这也是为数不多的有实际用途的项目之一。

和DNS一样，ENS注册并不是终身有效，而是按年交费。这倒不是什么大问题。相反，能每年持续地获得收入，才能持续地运营下去。

除了ETH地址，ENS现在也可以解析BTC、LTC和狗狗币地址，这让它有望成为一个跨链解析的标准。

我强烈建议使用ETH的用户去注册一个属于个人的ENS域名，因为目前仅有40万个名字已被注册，所以可选择的余地非常大。

最后，请务必注意：本文推荐注册一个个人专属的ENS域名，但强烈不建议以任何方式炒作ENS Token。
